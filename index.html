Spring Boot Support There are four entries in Spring Initializr for Spring Cloud Azure: * Azure support * Spring Boot Starter for [Azure Active Directory](https://azure.microsoft.com/services/active-directory/) * Spring Boot Starter for [Azure Key Vault](https://azure.microsoft.com/services/key-vault/) * Spring Boot Starter for [Azure Storage](https://azure.microsoft.com/services/storage/) You can also get starters for the following services from the [Azure Spring Boot repo](https://github.com/Microsoft/azure-spring-boot) on GitHub: * Spring Boot Starter for [Azure Active Directory B2C](https://azure.microsoft.com/services/active-directory-b2c/) * Spring Boot Starter for [Azure Cosmos DB](https://azure.microsoft.com/services/cosmos-db/) * Spring Boot Starter for [Azure Service Bus](https://azure.microsoft.com/services/service-bus/) * Spring Boot Starter for [Azure Service Bus](https://azure.microsoft.com/services/service-bus/) with JMS * Spring Boot Starter for Spring Data Gremlin with [Azure Cosmos DB](https://azure.microsoft.com/services/cosmos-db/) and any Gremlin-compatible data store * Spring Boot Starter for [Micrometer with Application Insights](https://docs.microsoft.com/azure/azure-monitor/app/micrometer-java) * Spring Boot Starter for [Azure Media Services](https://azure.microsoft.com/services/media-services/) The Azure Support entry contains auto-configuration support for Azure managed services. It also provides the capability to automatically inject credentials from Cloud Foundry into your applications using Azure managed services. Let us take Spring Boot Starter for Azure Service Bus as an example. You can add the usual azure-servicebus dependency to your project and the Spring Boot auto-configuration will automatically configure your Spring application with Azure Service Bus. 
<dependency> 
  <groupId>com.microsoft.azure</groupId>
  <artifactId>azure-servicebus</artifactId>
</dependency>
Alternatively, you can add the full name of the Spring Boot Starter for Azure Service Bus. 
<dependency>
  <groupId>com.microsoft.azure</groupId>
  <artifactId>azure-servicebus-spring-boot-starter</artifactId>
</dependency>
